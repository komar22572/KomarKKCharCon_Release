# KomarKKCharCon_Release
Public version of CharacterController plugin for KoboldKare
Current ver - 1.4.1
Edit your kobold + use many fun and useful functions.
Create and apply custom stats and clothing presets.
To use all of the features you either need to be the host or on server with (C) in the name.
Change config file for custom controls, it can be found in game_folder\BepInEx\config
As default: f5 for toggle plugin panel.
